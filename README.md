Python Text Analytics & Excel Automation Project

## Project Overview

This project uses Python to read, clean, organize, and analyze text-based data from two files: one containing ranked novels and another containing customer product reviews. The program parses the raw text into Python dictionaries and lists, calculates review-level statistics, performs basic sentiment classification, and exports the final results into a professionally formatted Excel workbook.

## Business Purpose

The goal of this project is to demonstrate how Python can automate data preparation and reporting tasks that would otherwise require manual work in Excel. The final workbook provides a clean, organized view of review data, novel statistics, formulas, tables, and summary information.

## Tools Used

* Python
* OpenPyXL
* Excel
* Regular Expressions
* Dictionaries and Lists
* Text Analysis
* Data Cleaning

## Key Features

* Reads and processes raw `.txt` files
* Converts review data into a list of dictionaries
* Converts novel ranking data into a dictionary grouped by author
* Calculates review character count
* Calculates review word count
* Adds an Excel formula for average word length
* Applies basic rule-based sentiment classification
* Creates a formatted Excel workbook with multiple worksheets
* Includes tables, formulas, summary statistics, and professional formatting

## Output

The project creates a final Excel file called `data.xlsx` with the following worksheets:

1. Novel Statistics
2. Review Statistics
3. Novels by Author
4. Dictionary Preview
